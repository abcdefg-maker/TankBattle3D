# TankBattle3DD - 3D坦克大战 游戏开发策划书

## 一、项目概述

### 1.1 项目名称
TankBattle3DD（3D坦克大战）

### 1.2 项目简介
基于经典FC坦克大战（Battle City）的玩法，使用Three.js开发的3D版本坦克大战游戏。采用第三人称视角，所有模型由基本3D几何图形（立方体、圆柱体、球体等）拼凑而成，在浏览器中即可运行。

### 1.3 技术栈
- **渲染引擎**: Three.js（通过CDN引入）
- **语言**: 原生JavaScript (ES6+)
- **运行环境**: 现代浏览器（Chrome/Firefox/Safari/Edge）
- **无需构建工具**: 纯前端静态文件，打开HTML即可运行

## 二、核心玩法设计

### 2.1 游戏模式
- **单人闯关模式**: 玩家操控坦克，消灭所有敌方坦克过关
- **关卡制**: 共设计3个关卡，难度递增
- **胜利条件**: 消灭当前关卡所有敌方坦克
- **失败条件**: 玩家坦克被摧毁 或 基地被摧毁

### 2.2 操作方式
| 按键 | 功能 |
|------|------|
| W / 上箭头 | 前进 |
| S / 下箭头 | 后退 |
| A / 左箭头 | 左转 |
| D / 右箭头 | 右转 |
| 空格键 | 发射子弹 |
| 鼠标移动 | 控制炮塔旋转方向 |

### 2.3 第三人称相机
- 相机位于玩家坦克后上方，形成经典的第三人称追踪视角
- 相机跟随坦克移动和旋转
- 相机与坦克之间保持固定距离和角度
- 相机平滑跟随，带有轻微延迟以增强体验

## 三、游戏元素设计

### 3.1 坦克模型（几何图形拼装）

#### 玩家坦克（绿色系）
- **车身**: 扁平长方体（BoxGeometry）- 主体
- **炮塔**: 小型立方体（BoxGeometry）- 位于车身上方
- **炮管**: 圆柱体（CylinderGeometry）- 从炮塔前方伸出
- **履带**: 两侧各一个扁长方体（BoxGeometry）- 深色
- **颜色**: 军绿色 (#4a7c59)

#### 敌方坦克（红色系/灰色系）
- 结构与玩家坦克相同，颜色区分
- **普通敌人**: 灰色 (#888888)
- **精英敌人**: 红色 (#cc3333)，移动速度更快
- **重型敌人**: 深红色 (#991111)，需要2发子弹才能摧毁

### 3.2 地图元素

#### 砖墙
- 立方体组合，棕红色 (#8B4513)
- 可被子弹摧毁（1发击碎）

#### 钢墙
- 立方体，银灰色带金属光泽 (#C0C0C0)
- 不可摧毁，子弹反弹

#### 水域
- 扁平面，蓝色半透明 (#4488ff)
- 坦克无法通过，子弹可飞越

#### 草丛
- 扁平面，绿色半透明 (#33aa33)
- 坦克和子弹都可通过，提供视觉遮蔽

#### 基地（老鹰）
- 由几何体拼成的简易鹰形标志
- 位于地图底部中央
- 被击中则游戏失败

### 3.3 子弹
- 小球体（SphereGeometry）
- 黄色发光效果
- 直线飞行，碰到障碍物或坦克后消失

### 3.4 道具系统（击杀敌人随机掉落）
- **星星（加速）**: 黄色八面体，提升移动速度5秒
- **盾牌（护盾）**: 蓝色环形，无敌3秒
- **炸弹（全屏清除）**: 红色二十面体，消灭所有当前敌人

## 四、关卡设计

### 4.1 关卡1 - 新手训练
- 地图: 20x20格子，少量砖墙
- 敌人: 4个普通坦克
- 特点: 简单布局，让玩家熟悉操作

### 4.2 关卡2 - 城市作战
- 地图: 20x20格子，大量砖墙和钢墙构成巷道
- 敌人: 3个普通坦克 + 2个精英坦克
- 特点: 复杂地形，需要利用掩体

### 4.3 关卡3 - 最终决战
- 地图: 20x20格子，混合地形（砖墙、钢墙、水域、草丛）
- 敌人: 3个普通 + 2个精英 + 2个重型坦克
- 特点: 高难度，全面考验

---

## 五、AI设计

### 5.1 敌方坦克AI
- **巡逻模式**: 随机方向移动，遇到障碍物转向
- **追踪模式**: 当玩家进入感知范围时，朝玩家方向移动
- **攻击模式**: 面朝玩家时自动开火
- **优先级**: 50%概率朝基地方向移动，50%概率追踪玩家

### 5.2 敌人生成规则
- 从地图顶部的3个出生点轮流生成
- 每次最多同时存在4个敌人
- 生成间隔: 3秒

## 六、视觉效果

### 6.1 光照
- 环境光：柔和的全局照明
- 方向光：模拟太阳光，产生阴影
- 点光源：爆炸时的动态光效

### 6.2 特效
- **爆炸效果**: 粒子系统 + 橙色光球扩散
- **开火闪光**: 炮口短暂白色闪光
- **坦克移动**: 履带灰尘粒子效果（简化版）

### 6.3 地面
- 大平面作为地面，深绿色
- 网格纹理模拟地面方格

## 七、UI界面

### 7.1 游戏HUD
- 左上角: 玩家生命值（心形图标 x 3）
- 右上角: 当前分数
- 右侧: 剩余敌人数量指示
- 底部中央: 当前关卡

### 7.2 菜单界面
- **开始界面**: 游戏标题 + 开始按钮
- **暂停界面**: ESC暂停，显示继续/重新开始选项
- **结算界面**: 显示得分、击杀数、评价
